## Why These Prompts?

1. **Hyper-specific** — Traditional LLMs hesitate and suggest 10+ options
2. **Multi-constraint** — Budget + geo + timing + tech stack + skill level
3. **Niche contexts** — Not in standard training data
4. **Complete stacks** — Requires combining API + framework + hosting + auth
5. **Immediate "wow"** — Jury can see the precision vs traditional LLM exploration
